# ⚡ VoltStream | Smart Energy Dashboard

**VoltStream** is a mobile-first, real-time energy monitoring prototype designed to bridge the gap between complex electrical usage data and household consumers. It turns boring meter readings into an interactive, visual experience.

## 🌟 Overview
As an Electrical Engineering student, I noticed that most people don't understand their energy consumption until a high bill arrives. VoltStream was built to make energy "visible." It allows users to simulate room-by-room consumption, see live tariff estimates, and understand their load profile through interactive data visualization.

**This project was a featured highlight at the [Insert College Name] Technical Expo, recognized by faculty for its multidisciplinary approach to Power Systems and Web Development.**

## 📱 Features
- **Mobile-First Design:** Built with a "Glassmorphic" dark theme optimized for smartphone screens using CSS Flexbox and Grid.
- **Real-Time Load Profiling:** Interactive sliders simulate the power draw (Watts) of individual rooms, with instant updates to the total units and estimated bill.
- **Dynamic Data Visualization:** Uses **Chart.js** to render a live energy curve that reacts immediately to user input.
- **Smart Toggles:** Simulates circuit-level control, allowing users to toggle room loads on/off to see the immediate impact on their daily and weekly costs.
- **AI-Logic Engine:** A threshold-based suggestion system that provides "AI Insights" when consumption patterns exceed safety or budgetary limits.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Custom Variables, Flexbox, Transitions)
- **Logic:** Vanilla JavaScript (ES6+)
- **Visualization:** Chart.js
- **Icons:** FontAwesome 6.0

## 🚀 Future Roadmap (AI & IoT)
VoltStream is currently a high-fidelity prototype. My future plans for this project include:
1. **IoT Integration:** Connecting the dashboard to ESP32 microcontrollers and CT sensors for real-time hardware data logging.
2. **Predictive ML Models:** Implementing a regression model to predict monthly bills based on the first 7 days of usage.
3. **Anomaly Detection:** Using AI to identify the "electrical signature" of faulty appliances to prevent short circuits and energy waste.
